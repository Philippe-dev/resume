Adaptation du thème Resume par Start Bootstrap. Basé sur Bootstrap 5.

## Menu

Le menu latéral est géré par le plugin Simple Menu

## Configurateur

Le configurateur permet de choisir :

- la couleur principale du blog : barre latérale, liens et boutons
- l'image de profil, de préférence au format carré, taille moyenne
- les liens vers les réseaux sociaux
